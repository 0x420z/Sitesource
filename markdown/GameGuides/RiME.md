##Additional Notes

* Cheat Table Notes
    
    1. Camera can be enabled and disabled at any time with [Page Down]. Controlled with the keypad keys as shown in the table.
    2. Enable  Timescale control with the Pause script by pressing [P].

        Key | Command
        --|--
        P | Pause / Unpause
        F5 | 0.6 normal time
        F6 | 0.4 normal time
        F7 | 0.2 normal time
        F8 | 0.1 normal time
